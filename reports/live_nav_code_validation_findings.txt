
LIVE NAV FETCH - DATA SOURCE VALIDATION FINDINGS
=================================================

Objective: Fetch live NAV data from mfapi.in for 6 AMFI scheme codes referenced
in the assignment (125497, 119551, 120503, 118632, 119092, 120841).

Finding 1 - Internal consistency:
  All AMFI codes referenced in the assignment brief exactly match the codes
  present in fund_master.csv.

Finding 2 - External mismatch with live registry:
  When queried against the live mfapi.in API, only 1 of 6 (118632, Nippon
  India Large Cap) returned the expected fund. The remaining 5 codes
  returned unrelated schemes.

Conclusion:
  The provided CSV datasets are simulated/practice data and not a live
  extract of the current AMFI registry. AMFI codes get reassigned over
  time, so a code valid at one point may point to a different fund today.

Recommendation:
  Use fund_master.csv as the source of truth for AMFI code -> fund name
  mapping throughout this project, not the live API's current mapping.
